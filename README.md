Digital Clock ⏰

📌 Description

This is a simple Digital Clock project built using HTML, CSS, and JavaScript.
The clock displays the current time and automatically updates every second using JavaScript.

🚀 Features

- Shows the current time in real-time
- Updates every second
- Simple and clean user interface
- Built with pure JavaScript (no libraries)

🛠 Technologies Used

- HTML
- CSS
- JavaScript

▶️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Run the index.html file in your web browser.
